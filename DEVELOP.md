Usage: docusaurus swizzle [options] [themeName] [componentName] [siteDir]

Wraps or ejects the original theme files into website folder for customization.

Options:
  -w, --wrap        Creates a wrapper around the original theme component.
                    Allows rendering other components before/after the original theme component.
  -e, --eject       Ejects the full source code of the original theme component.
                    Allows overriding the original component entirely with your own UI and logic.
  -l, --list        only list the available themes/components without further prompting (default: false)
  -t, --typescript  copy TypeScript theme files when possible (default: false)
  --danger          enable swizzle for unsafe component of themes
  -h, --help        display help for command
